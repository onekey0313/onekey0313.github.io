Hexo搭建blog优化篇

主题配置文件

blog(自己建博客的文件夹)/source/_data/next.yml

其他配置项

为网页标题添加崩溃欺骗搞怪特效

  • 创建JS文件：

xxx/themes/next/source/js/src/crash_cheat.js

  • 在布局layout文件中引用JS文件：

xxx/themes/next/layout/_layout.swig

为博客添加点击爱心效果

  • 创建JS文件：

    xxx/themes/next/source/js/src/click_love.js

  • 在布局layout文件中引用JS文件：

    themes/next/layout/_layout.swig

配置文章底部标签

  • 文章底部默认是# 标签名, 此处修改为FontAwesome字体图标：

xxx/themes/next/layout/_macro/post.swig

添加自定义样式文件

  • 在_data目录添加自定义样式文件：

    xxx/source/_data/styles.styl

  • 配置NexT主题使用此样式文件：

    source/_data/next.yml

添加自定义变量文件

  • 在_data目录添加自定义变量文件：

    xxx/source/_data/variables.styl

  • 配置NexT主题使用此样式文件：

    xxx/source/_data/next.yml

站点底部添加访客量和访问量

xxx/source/_data/next.yml

为每篇文章添加阅读次数

  • 使用LeanCloud来扩展此功能，此处需要注册并创建数据库，数据库名称为Counter。

  • 注册LeanCloud，允许使用Github绑定登录，创建数据库：

    登录账号 —> 进入控制台 —> 创建新应用 —> 点击存储 —> 新建 Class —> Counter

  • 获取应用的ID和Key：

登录账号 —> 进入控制台 —> 点击应用 —> 点击设置 —> 应用 Key —> 记录 ID 和 Key

  • 由于NexT主题已集成，直接修改配置文件即可：

    source/_data/next.yml

  • 在本地预览时，无法显示阅读次数，部署到服务器即可正常显示了。

为博客添加Valine极简评论

  • 此处依旧是使用LeanCloud来扩展此功能，需要应用的ID和Key：

登录账号 —> 进入控制台 —> 点击应用 —> 点击设置 —> 应用 Key —> 记录 ID 和 Key

  • 由于NexT主题已集成Valine，直接修改配置文件即可：

source/_data/next.yml
